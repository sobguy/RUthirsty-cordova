    inset 0 -1px 0 0 rgba(255, 255, 255, 0.05);

按钮有5个不同的状态，平滑过渡:
- 空闲（闪光）

3. 点阵图案（微妙）
4. 浮动粒子（慢动作）

### 4. 智能动画时序
错开的动画创建编排体验:
- 标题: 0秒延迟
- 计数器: 0秒延迟
- 按钮: 0.2秒延迟
