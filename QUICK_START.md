## 🎯 常见问题

### Q: 如何清空所有数据？

**A**: 在浏览器控制台执行：
```javascript
localStorage.clear();
location.reload();
```

---

### Q: 如何修改每次记录的水量？

**A**: 编辑 `ruthirsty/www/js/dataManager.js`：
```javascript
AMOUNT_PER_GLASS: 250, // 改为你想要的毫升数
```

---

### Q: 如何更改数据保留天数？

**A**: 编辑 `ruthirsty/www/js/app.js`：
```javascript
DataManager.cleanupOldRecords(30); // 改为你想要的天数
```

---

### Q: 动画太多，如何关闭？

**A**: 在操作系统设置中启用"减少动画"选项，应用会自动适配。

---

### Q: 如何构建 Android APK？

**A**: 需要先配置 Android SDK：
```bash
export ANDROID_HOME=/path/to/android/sdk
cd ruthirsty
cordova build android
```
